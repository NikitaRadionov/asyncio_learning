asyncio это библиотека для написания конкурентного кода с использованием синтаксиса async/await

asyncio используется в качестве фундамента для нескольких асинхронных Python фреймворков, которые предоставляют высоко производительные сети
и веб-серверы, библиотеки для соединения с базой данных (для работы с базой данных), распределенные task queues, и т.д

asyncio часто идеально соответствует для IO-bound и высокоуровнего структурированного newtwork кода.

asyncio предоставляет набор высокоуровнего API для:
 1. запуска Python корутин конкурентно и иметь полный контроль во время их исполнения
 2. выполнения newtwork IO и IPC
 3. контролирования подпроцессов (subprocesses)
 4. распеределения задач с помощью queues
 5. синхронизации конкурентного кода

 Кроме того, предоставляется набор низкогоуровненго API для разработчиков библиотек и фреймворков для:
  1. создания и управления циклов событий (event loops), которые обеспечивают асинхронное API для сети, запуск подпроцессов, манипулирование OS сигналами (signals), и тд
  2. реализации эффективных протоколов, используя протоколы транспортного уровня.
  3. связывания callback-based библиотек и кода с синтаксисом async/await
